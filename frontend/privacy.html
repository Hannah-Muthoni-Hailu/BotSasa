1. Who We Are

2. Information We Collect

Account Information

  • Name
  • Email address
  • Password (hashed using bcrypt)

API Keys

Each account receives a unique API key. Keys may be revoked. To obtain a new key, users must revoke the existing key, settle outstanding charges, and generate a replacement.

Context Documents

Users may upload text documents used to answer chatbot questions. Deleted documents are permanently removed from our MongoDB database and are not backed up.

Usage Information

  • Token usage
  • API key creation date
  • Billing cycle dates

We do not collect IP addresses, browser information, device information, API request logs, or error logs.

Cookies

Cookies are used only for authentication and session management.

3. Chat Conversations and AI Processing

BotSasa processes conversations through the OpenAI API. BotSasa does not store chatbot conversations. Conversation data is transmitted to OpenAI to generate responses. BotSasa has enabled OpenAI data-sharing features, meaning conversation data may be retained and processed by OpenAI according to OpenAI's applicable policies. BotSasa cannot delete conversation data retained by OpenAI.

4. Payment Information

BotSasa is a paid service. Payments are processed exclusively by Paystack (a Stripe company). BotSasa does not store payment card information.

5. How We Use Your Information

6. Information Sharing

We do not sell personal information. We share information only with trusted providers including OpenAI, Paystack, and Render where necessary to operate the service.

7. Data Retention

Account information is retained while your account is active. Upon verified deletion requests, account information, context documents, and API keys are removed, except where records must be retained by law. Conversation data retained by OpenAI is governed by OpenAI's policies.

8. Account Deletion

Request deletion by emailing support@your-domain.com. After verification, BotSasa will delete personal data under its control except where retention is legally required.
